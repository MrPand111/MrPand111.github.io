html中meta的使用

发表于 2020-03-09 更新于 2020-03-10 分类于 前端 阅读次数：

meta标签

  1. 提供的信息用户不可见，但却是文档的最基本的元数据

  2. 元数据是用来描述数据的一些基本数据。对于网页来说，meta就算通过一些字段信息来描述一下当前网页

http-equiv属性

  • 相当于http的文件头作用，它可以向浏览器传回一些有用的信息，以帮助正确和精确地显示网页内容

  • <meta http-equiv="参数" content="参数变量值">

  1. 用于设定网页的到期时间。一旦网页过期，必须到服务器上重新传输

  2. 告诉浏览器在【数字】秒后跳转到【一个网址】

  3. 设定页面使用的字符集

  4. 禁止浏览器从本地计算机的缓存中访问页面内容。访问者将无法脱机浏览

  5. 用来防止别人通过frame或iframe标签调用自己的页面

  6. 强制浏览器按照特定的版本进行渲染。如果是ie浏览器就用最新的ie渲染，如果是双核浏览器就用chrome内核

  7. 设置Cookie，如果网页过期，那么存盘的cookie将被删除

name属性

  • 主要用于描述网页，与之对应的属性为content，里面的内容主要是便于搜索引擎机器人查找信息和分类信息用的

  • <meta name="参数" content="具体的参数值">

  • 最重要的就是对keywords和description的设置。因为按照搜索引擎的工作原理，搜索引擎首先排除机器人自动检索页面中的keywords和description，并将其加入到自己的数据库，然后再根据关键词的密度将网站排序

  1. 在移动设备浏览器上，禁用缩放（zooming）功能，用户只能滚动屏幕

  2. 告诉搜索引擎，当前页面的主要内容是xxx

  3. 告诉搜索引擎，当前页面的关键字

  4. 告诉搜索引擎，标注网站作者是谁

  5. 标注网站的版权信息
